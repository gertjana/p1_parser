# P1Parser

Parses telegram's as they are output through the P1 serial port of a smartmeter

## Installation

```elixir
def deps do
  [
    {:p1_parser, "~> 0.1.3"}
  ]
end
```

## Documentation 

The docs can be found at [https://hexdocs.pm/p1_parser](https://hexdocs.pm/p1_parser).
